# rcss-fuzzy-agent

## Requirements

node v23.10.0

## Run

Install and setup rcssserver (used version 19.0.0)

https://github.com/rcsoccersim/rcssserver/releases

Install and setup rcssmonitor (used version 19.0.1)

https://github.com/rcsoccersim/rcssmonitor/releases

Run in terminals:

rcssserver

rcssmonitor

run.sh
